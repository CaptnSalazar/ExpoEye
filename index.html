Quick Overview

ExpoEye combines local OCR with lightweight scene classification and an LLM backend to provide contextual, actionable responses for event attendees. Designed for mobile-first use and hackathon demos.

Core Capabilities

  • OCR extraction from images (badges, maps, screens)
  • Scene classification (networking, navigation, tech glitch)
  • LLM-driven suggestions (Grok API integration or demo fallback)
  • Easy deployment via Streamlit or Docker

How to run locally

git clone https://github.com/your-username/ExpoEye.git
cd ExpoEye
python -m venv expoeye-env
source expoeye-env/bin/activate
pip install -r requirements.txt
streamlit run app.py

Set GROK_API_KEY in .env or use demo mode.
